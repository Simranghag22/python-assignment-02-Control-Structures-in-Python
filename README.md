# python-assignment-02-Control-Structures-in-Python

This repository contains Python programs demonstrating the use of conditional statements and loops.

---
#Task 1: Check if a Number is Even or Odd

### Description
This program:
- Takes an integer input from the user
- Uses an `if-else` statement
- Determines whether the number is even or odd
- Displays the result

### Concepts Used
- User input
- Conditional statements (`if-else`)
- Modulus operator (`%`)
- Formatted strings (f-strings)

Task 2: Sum of Integers from 1 to 50

### Description
This program:
- Uses a `for` loop
- Iterates from 1 to 50
- Calculates the total sum of all integers in the range
- Displays the final result

### Concepts Used
- `for` loop
- `range()` function
- Accumulator variable
- Arithmetic operations
